# cv-repo

This is an HTML and CSS based CV(resume) project. It includes my about, education details, project details,
experience details, along with my socials links. It also caters flexbox and some animations as well. It
also allows the user to toggle between dark mode and light mode, download the CV and send a message to me.
It has a responsive design that allows smooth user experience across different screens.
The project is mostly based on HTML and CSS, with a little additional JS for handling mode change and contact messages.

The project is live at: https://cv-repo-one.vercel.app/?authuser=0
